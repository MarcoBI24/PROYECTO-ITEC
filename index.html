¿Qué es HTML?

Es el Lenguaje de Marcado de Hipertexto (en siglas ), con HTML podemos estructurar todo una pagina web y sus elementos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos. Y para poderle darle estilo lo hariamos conectando HTML con CSS.
